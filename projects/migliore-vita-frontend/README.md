# MiglioreVita Frontend

## Setup
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start dev server:
   ```sh
   npm run dev
   ```

## Structure
```
/src
  ├── App.jsx          # Main app router
  ├── main.jsx        # React entry point
  ├── pages/          # Route components
  └── components/     # Reusable UI
```

## Pages
- `/` - Home Dashboard
- `/inventory` - Inventory Management
- `/sales` - Sales Dashboard
